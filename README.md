 Welcome to Ayocrypt whisper echo project
Ayocrypt Whisper Echo is an AI-powered voice interface that uses OpenAI Whisper for real-time speech recognition and a custom encryption engine to process and protect sensitive voice data. The app captures audio commands, transcribes them, encrypts the data, and provides secure output â€” ideal for hands-free cryptographic operations and secure communications.

Features
 Speech-to-text via OpenAI Whisper
 Text encryption and decryption engine (Ayocrypt)
 Voice command interface
AI-powered text processing
 Python-based with simple command-line control
